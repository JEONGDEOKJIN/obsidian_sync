
## DOM 이란 
``` bash 
- DOM 이란 
	- 내 생각: 'document'를 '토큰화' 시켜 'key : value 의 쌍'으로 표현한 '객체' 를 바탕으로 -> 'node 를 만들고 -> ' '부자 관계' 를 반영하여 '트리 자료구조' 로 표현한 것. 
	- 'HTML 문서의 계층적 구조와 정보' 를 표현하며, 이를 '제어할 수 있는 API' 즉, 프로퍼티와 메서드를 제공하는 트리 자료 구조 이다. (모던 자바스크립트 p677)
```
