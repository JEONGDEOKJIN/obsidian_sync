### await 
``` bash 
- await 는 반드시 promise 앞에서 사용해야 함 (출처 : 모던 자바스크립트 p880)
- await 는 비동기 처리 코드를 만났을 때, 다음 코드를 실행시키지 않고, 비동기 처리 상태가 settled 될 때 까지 대기 한다. -> 그리고 settled 가 되면, 다음 코드를 실행시킨다. 

- '처리 순서를 보장' 해야 할 경우, 모든 프로미스에 await 를 붙인다. 
- '처리 순서를 보장하지 않을 경우', promise.all 을 사용한다. (출처 : 모던 자바스크립트 p881) ⭐⭐⭐ 

- 에러 처리 
	- try - catch 사용 
	- catch 후속처리문을 사용할 수도 있음. 

```
